Инструкция
  1. Выберите проект, в который хотите загрузить файлы
  2. Нажмите "Выберите файлы" и выберите несколько файлов (удерживайте Ctrl для выбора нескольких)
  3. Добавьте описание (необязательно)
  4. Нажмите "Загрузить файлы"
Совет: Вы можете выбрать до 10 файлов одновременно. Каждый файл будет создан как отдельная запись в контенте проекта.
